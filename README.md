# Aaron Yuson Portfolio 2026

Interactive portfolio website for Aaron Yuson, a multimedia designer specializing in 3D modeling, branding, graphic design, web development, and motion design.

## Local Preview

The site can be opened directly through `index.html`. Use a local HTTP server when testing the 3D model viewer.

## Deployment

This is a static website configured for Vercel. See `DEPLOYMENT.md` for deployment and domain setup notes.
